# 知识层

本目录存放 M4 后续真模型写稿、三审和复盘可读取的正式知识素材。

素材以「精选速查版」平移自外部 MIT 项目（主要是 oh-story-claudecode），
保留来源、来源 ref、许可证和文件哈希，记录在 `_manifest.json`。CI 只信本目录，
不把 ignored 的 `Dev/` 参考仓当隐式输入。

平移原则：
- **速查而非全文**：每篇提炼决策路由 + 核心法则，避免知识层臃肿；真模型按需读。
- **MIT 署名随平移**：每篇 front matter 带 source/source_ref/license，许可全文见 `许可/`。
- **GPL 不平移**：GPL 项目（如 webnovel-writer）仅作架构思想参考，不搬代码或文本。
