# 题材路由速查

建书选题材、写大纲定位赛道时对照。按故事核心定位框架，严格执行结构比例与情绪节拍。
正文细节仍以本书大纲、账本为准。

## 大类路由

| 故事核心 | 框架 | 关键备注 |
|---|---|---|
| 男人伤害女人，离开后悔追不回 | 追妻火葬场 | 女主离开后不回头是核心 |
| 前世惨死，重生利用信息差复仇 | 重生复仇 | 反派需分层嵌套 |
| 伴侣出轨/算计，觉醒独立 | 婚恋/小三 | 发现过程要有悬念 |
| 恶霸欺压老实人，恶有恶报 | 世情 | 靠细节不靠大事件 |
| 女主被虐至死，男主死后追悔 | 死人文学 | 核心是「来不及」 |
| 特殊相遇，男主追求宠溺 | 霸总/甜宠 | 甜的密度决定粘性 |
| 底层崛起，升级打怪（都市+武道） | 都市高武 | 金钱驱动 + 分层地图 |
| 废材/落魄天才，修仙登顶 | 仙侠/玄幻 | 力量体系必须清晰 |

## 选型原则

- 先判大类，再看具体变体；核心冲突定框架，不由标签定。
- 力量体系类（玄幻/高武）：升级阶梯与地图分层必须开篇就立住。
- 情感类（言情/虐恋）：情绪节拍密度优先于情节复杂度。
- 爽文类：打脸/兑现节奏要稳，铺垫比打脸更重要。

## 写作公式骨架

- 现代复仇/打脸：6 章短篇结构，开篇低位 → 压抑积蓄 → 兑现爆发。
- 都市系统/逆袭：前 3 章立系统、立金手指、立第一个小爽点。
- 玄幻/重生逆袭：19 章大结构，每 3-5 章一个小高潮，卷末大高潮。
- 详细公式见原始素材 genre-writing-formulas.md（按题材套用结构骨架）。
