# 人物与对话技法速查

设计人物、写对话、处理人物关系时加载。合并自 character-design-methods 与 dialogue-mastery。
正文细节仍以本书大纲、账本、名册为准。

## 人物设计三维度

1. **核心欲望**：角色最想要什么？欲望驱动一切行动，没有欲望的角色是工具人。
2. **致命缺陷**：阻碍欲望实现的内在缺陷（骄傲/懦弱/执念），是成长弧的起点。
3. **标志性细节**：一个让读者记住的外化细节（口头禅/小动作/物件），比大段外貌描写有效。

## 人物弧线

- 起点：缺陷 + 错误信念 → 经历：挑战 + 代价 → 终点：觉悟 + 改变（或悲剧性坚持）。
- 弧线要付代价：没有代价的成长不可信。
- 反派也要有欲望和逻辑：纯恶的工具人反派无法支撑长篇。

## 对话技法

- **潜台词**：角色说的不是想的；好对话有「水面下的意思」。
- **冲突即对话**：每段对话应有权力/信息/情感的错位，纯信息传递的对话是废话。
- **标签克制**：多用动作替代「他说/她问」；对话标签占比高 = 机器味重。
- **声音区分**：不同角色的用词、句长、语气应可辨，不能换名字就一样。

## 关系网

- 每对核心关系都要有张力来源（利益/情感/信息差）。
- 关系变化要有事件驱动，不能靠旁白「他们关系变好了」。
- 名册里的专名要与关系网一致，设定校对会查。

## 自检

- [ ] 这个角色有核心欲望吗？（不是只有功能）
- [ ] 对话有潜台词/冲突吗？（不是只传信息）
- [ ] 对话标签是否过多？（文风铁律会查占比）
