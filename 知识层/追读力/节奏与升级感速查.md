# 节奏与升级感速查

Phase 3-4 节奏把控、升级感设计、期待感管理时加载。正文细节仍以本书大纲、账本为准。

## 路由

| 在设计什么 | 方法 |
|---|---|
| 升级感 / 升级节奏 | 升级感三步法 + 循环嵌套 + 对比表 |
| 情绪节奏 | 情绪模块公式 + 桥段结构化 |
| 高潮设计 | 高潮分类 + 反推四要素 |
| 开篇节奏 | 黄金五章 + 开头精简法 + 节奏底线 |
| 期待感管理 | 拉期待原则 + 真空期急救 |
| 战斗 / 智斗 | 回合制卡牌 + 三步两线法 |

## 升级感三步法

1. **确立参照系**：读者要知道「现在多强」——给量化锚点（排名/境界/财富对比）。
2. **制造落差**：新场景/新对手打破旧参照系，让读者感到「原来还有更强的」。
3. **兑现跨越**：角色跨过落差，参照系更新；循环嵌套，每轮落差递增。

## 期待感管理

- **拉期待**：先让读者知道「可能得到什么」，再延迟兑现；期待值 = 价值 × 可能性。
- **真空期急救**：主线推进间隙用副线/日常/伏笔填，但不能替代推进。
- **章末钉问题**：每章结尾钉住一个「下章必须回答」的问题。

## 节奏底线

- 每章必须有局势变化（钩子不能替代推进）。
- 黄金五章：前 5 章立人设、立金手指、立第一个爽点、立核心冲突。
- 高潮反推：先定高潮要解决什么，再倒推需要哪些铺垫。
