# 反转设计速查

设计反转、打脸节奏、误导嵌套时加载。反转类型与拆文 meta 的 reversal_type 字段一致。
正文细节仍以本书大纲、账本为准。

## 类型路由

| 要设计什么 | 反转类型 | 关键操作 |
|---|---|---|
| 角色不是你以为的人 | 身份反转 | 埋 3 处行为细节暗示 |
| 真相从另一角度完全不同 | 视角反转 | 引入另一角色视角打破认知 |
| 做事的原因不是你以为的 | 动机反转 | 高压场景中二选一暴露真动机 |
| 时间顺序与读者理解不同 | 时间线反转 | 不撒谎只调叙述顺序 |
| 读者掌握了错误信息 | 信息反转 | 新证据直接否定旧「事实」 |
| 对角色/关系的理解被颠覆 | 认知反转 | 全程铺感情色彩，结尾翻转 |
| 甜宠/喜剧/报应型 | 无反转 | 走报应兑现或甜度递进，不硬塞 |
| 要叠多层 | 嵌套反转 | 主次分明，间隔递减 |

## 设置与揭示

- **埋线**：反转前必须有至少 2-3 处「事后回看才懂」的细节，不能凭空翻。
- **误导**：给读者一个「看起来更合理」的错误解释，真相反着走。
- **揭示**：新证据一次性推翻旧认知，不挤牙膏；揭示后立刻有情绪兑现。

## 打脸节奏

- 打脸前必须铺垫（道具/人物/实力就位），铺垫比打脸更重要。
- 打脸要分层：小打脸密集（每 1-2 章），大打脸间隔（卷末）。
- 报应兑现型：恶有恶报的兑现要具体、可感，不能靠旁白宣告。

## 自检

- [ ] 反转前有埋线细节吗？（至少 2-3 处）
- [ ] 反转后行为逻辑通顺吗？（不能只翻结论不翻动机）
- [ ] 揭示后有情绪兑现吗？（爽感/震惊/唏嘘）
