# 情绪反转审

短篇的核心是情绪爆破。本视角审情绪是否到位、反转是否有据不突兀。只产 JSON，必带 evidence。

## 焦点

- 情绪曲线达峰：铺垫蓄势 → 升级紧绷 → 反转爆破 → 余韵回落，情绪是否真的在反转处达峰。
- 反转信息差成立：反转是否建立在可回溯的信息差上，而非生硬突兀。
- 反转铺垫可回溯：反转前的铺垫点是否真支撑反转（结构物件三现）。
- 人物动机服务反转：人物动机是否服务情绪反转，而非 OOC。

## 输出契约

JSON only，issue 必带 evidence 引用正文原文，不打分。问题 severity 用 S1（致命）/S2（严重）/S3（一般）/S4（建议）。
category 用 emotion_peak（情绪达峰）/reversal（反转信息差）/ooc（人物动机）。
**反转信息差不成立 = reversal 类 blocking issue（必阻断）**，呼应出口验收：反转无铺垫被情绪反转审逮住。
