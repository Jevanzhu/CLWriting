# CSV 上下文参考

本目录中的 CSV 是轻量参考表，用于人工或 Agent 快速取用创作提示。
它们不是数据库，也不作为项目事实真源。

## 使用规则

- 每次只读取与当前任务相关的 1-2 个表。
- 读取后要结合 `state.json`、`memory.json` 和设定集判断。
- CSV 中的建议不能覆盖用户明确设定。
- 所有条目都面向 1-10 万字短篇/中篇。
