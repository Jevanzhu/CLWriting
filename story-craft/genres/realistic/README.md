# realistic

## 定位

现实题材流派包，覆盖现实题材、都市日常、职场婚恋、年代等偏写实故事。
目标是让 1-10 万字短篇/中篇故事有具体处境、真实压力和可回收的生活细节。

## 适用题材

- 现实题材
- 都市日常
- 职场婚恋
- 年代
- 部分女频悬疑和家庭伦理故事

## 核心原则

- 用具体事件承载社会议题。
- 人物选择必须受经济、家庭、职业或制度成本限制。
- 细节必须改变关系、行动或结尾理解。
- 结尾允许有限胜利，不要求问题彻底解决。

## 推荐读取顺序

1. `patterns.md`：选择现实压力模型。
2. `checklist.md`：写前和审查时检查真实感。
3. `templates/genres/现实题材.md`：补充具体流派模板。
