# xuanhuan

## 定位

奇幻/仙侠流派包，覆盖修仙、玄幻、高武、西幻、都市异能、系统流等故事。
目标是把能力、规则和代价压缩为短篇/中篇可执行的行动机制。

## 适用题材

- 修仙
- 高武
- 西幻
- 都市异能
- 系统流
- 幻想言情中的能力规则

## 核心原则

- 能力规则少而清晰。
- 每次使用能力都要带来代价或新问题。
- 高潮解决方案必须包含主角选择。
- 不展开长线战力体系。

## 推荐读取顺序

1. `patterns.md`：选择能力规则模型。
2. `checklist.md`：检查能力边界和代价。
3. `templates/output/设定集-力量体系.md` 与 `设定集-金手指.md`：落地设定。
