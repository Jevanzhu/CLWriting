# zhihu-short

## 定位

知乎短篇流派包，覆盖高钩子、高密度、强反转和情绪兑现的短篇故事。
目标是在 1-5 万字短篇内完成清晰问题、压迫升级、证据反击和情绪释放；中篇使用时应分阶段反击，避免拉长受压段落。

## 适用题材

- 知乎短篇
- 家庭反击
- 职场打脸
- 情感反转
- 身份揭示
- 爽感型现实故事

## 核心原则

- 开篇直接进入冲突。
- 前 10% 必须抛出强问题。
- 背景信息只在反转需要时补。
- 反击必须来自前文证据或独特优势。
- 结尾要干净，不拖尾解释。

## 推荐读取顺序

1. `patterns.md`：选择钩子和反击结构。
2. `checklist.md`：检查密度、证据和结尾。
3. `templates/genres/知乎短篇.md`：补充题材模板。
