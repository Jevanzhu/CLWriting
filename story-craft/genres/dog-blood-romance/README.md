# dog-blood-romance

## 定位

情感冲突流派包，覆盖狗血言情、替身文、豪门总裁、现言脑洞、青春甜宠等故事。
目标是让短篇/中篇的情绪张力来自人物欲望、伤害和边界变化，而不是重复误会。

## 适用题材

- 狗血言情
- 替身文
- 豪门总裁
- 现言脑洞
- 青春甜宠
- 幻想言情和古言中的强关系线

## 核心原则

- 每次争吵都要改变关系权力。
- 误会必须来自人物缺陷或真实利益冲突。
- 甜点和虐点都要服务主线选择。
- 伤害不能被一句道歉自动抹平。

## 推荐读取顺序

1. `patterns.md`：选择关系拉扯模式。
2. `checklist.md`：检查情绪推进和边界。
3. `references/shared/naming-and-voice-gaps.md`：检查称呼和语调变化。
