# rules-mystery

## 定位

悬疑/规则怪谈流派包，覆盖悬疑灵异、规则怪谈、悬疑脑洞、女频悬疑等故事。
目标是让短篇/中篇谜题有清晰信号、误导、反转和回收。

## 适用题材

- 悬疑灵异
- 规则怪谈
- 悬疑脑洞
- 女频悬疑
- 克苏鲁小切口故事

## 核心原则

- 开篇给信号，不给答案。
- 每条线索必须有表层解释和真实含义。
- 规则不能随剧情临时改变。
- 行动线要闭合，主题层面可以留白。

## 推荐读取顺序

1. `patterns.md`：选择谜题或规则结构。
2. `checklist.md`：检查线索公平性和回收。
3. `references/short/plot-signal-vs-spoiler.md`：控制剧透边界。
