# period-drama

## 定位

历史/古风流派包，覆盖历史古代、历史脑洞、古言、宫斗宅斗、民国言情、年代等故事。
目标是让短篇/中篇的时代背景服务人物选择，而不是变成设定展示。

## 适用题材

- 历史古代
- 历史脑洞
- 古言
- 宫斗宅斗
- 民国言情
- 年代

## 核心原则

- 只写会改变行动的制度、礼法或时代压力。
- 称呼、物件和空间要体现身份关系。
- 古代/年代人物不能直接套现代口号。
- 结尾要承认时代限制。

## 推荐读取顺序

1. `patterns.md`：选择时代压力结构。
2. `checklist.md`：检查称呼、礼法、历史细节。
3. `references/shared/naming-and-voice-gaps.md`：校准语调。
